Version 1.0

  • Enhancement
    • "Run" slider: change of meaning to make it more intuitive. Previously adjustment of sleep period (faster on the left), now adjustement of speed (faster on the right)
    • Change some values of EDU opcodes
      • MOVA and MOVB before JMP and JMPZ (needed to comply with educational documents)
      • Opcode values added in EDU processor information
  • Bug Fixes
    • Enable "Save As" when a file is opened (can be saved with another file name)

Version 0.5

  • Bug Fixes
    • Memory content of address was displayed at address + 1

Version 0.4

  • General changes
    • Improvement of user interface: better window management (resizing), definition of tool tips.
    • Help added: displayed various helps (board, architecture, release notes) into browser.
    • Board: radio button to display addresses in hexadecimal.
    • Code re-design (to better support of new assembly language)
  • 6502
    • Fixes: errors in operand list for some opcodes, execution errors.
    • Important: testing of instructions still not exhaustive.

Version 0.3

  • General changes
    • New assembly language: 6502
  • Edu1
    • support of labels: before an instruction or on a separate line
